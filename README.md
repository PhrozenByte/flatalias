Flatalias 📦
============

[![GitHub release](https://img.shields.io/github/release/PhrozenByte/flatalias.svg)](https://github.com/PhrozenByte/flatalias/releases)
[![License](https://img.shields.io/github/license/PhrozenByte/flatalias.svg)](https://github.com/PhrozenByte/flatalias/blob/main/LICENSE)
[![Issues](https://img.shields.io/github/issues/PhrozenByte/flatalias.svg)](https://github.com/PhrozenByte/flatalias/issues)
[![Pull Requests](https://img.shields.io/github/issues-pr/PhrozenByte/flatalias.svg)](https://github.com/PhrozenByte/flatalias/pulls)

**Flatalias** is a lightweight CLI tool that generates short, convenient command aliases for installed Flatpak applications and makes them available in your shell. For example, if you have the Mozilla Firefox Flatpak installed, you no longer have to remember the full Flatpak ID and can launch it with `firefox` instead.
